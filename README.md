# 🏯 三国霸业

多人联机回合制策略网页游戏，2–4人在线对战，争夺三国天下。支持 AI 机器人、抽卡系统、技能卡、存档读档，手机电脑均可游玩。

## 如何开始

输入名号 → 进入大厅 → 创建房间或加入房间 → 等人齐了点"开始游戏"。

把房间号（4位字母数字）发给朋友即可联机。也可以添加 AI 机器人凑人数。

## 游戏规则

### 胜利条件
- 占领全部 24 座城池，直接获胜
- 30 分钟时间到后按得分结算：城池数占 70%，兵力占 30%

### 回合流程
每回合 60 秒，可执行以下操作（不限次数）：

| 操作 | 说明 |
|------|------|
| 🎴 五连抽卡 | 每回合免费抽一次，可获得武将、技能卡、金币或兵力 |
| ⚔️ 出征 | 选择出发城池、目标城池、武将和兵力，发起攻击 |
| 🔥 全军出击 | 一键派出全部兵力（自动保留 500 兵守城） |
| 🏰 征兵 | 花金币招募士兵，1000 兵 = 2000 金 |
| 📜 使用技能卡 | 使用抽到的技能卡获得战术优势 |
| ⏭️ 结束回合 | 主动结束，轮到下一位玩家 |

### 经济系统
- 初始金币 5000，开局每人分配一座主城（10000 兵）
- 每回合开始时：每座城池产出 1000 金，自动恢复 500 兵（上限 30000）

### 地图
24 座城池基于三国真实地理分布，城池之间有固定路线，只能沿路线攻击（除非使用特殊技能卡）。

### 战斗系统
- 出征后城池至少保留 500 兵力
- 武将提供三维加成：攻击力影响进攻，防御力影响守城，谋略减少己方兵损
- 战斗结果有 ±20% 随机浮动
- 攻城胜利后，武将自动驻守目标城池
- 击败敌方城池可俘获驻守武将（未战死的归入攻方）
- 战败方武将有概率战死：金卡 15%、紫卡 25%、蓝卡 40%

## 抽卡系统

每回合可免费五连抽一次。武将全局唯一，被抽走后其他玩家不会再抽到同名武将。

| 类型 | 概率 | 内容 |
|------|------|------|
| 🥇 金卡武将 | 8% | 吕布、关羽、马超、张飞、诸葛亮、曹操、孙权、刘备 |
| 🟣 紫卡武将 | 15% | 赵云、周瑜、典韦、司马懿、甘宁、太史慈 |
| 🔵 蓝卡武将 | 15% | 黄忠、孙策、张辽、徐晃、魏延、黄盖 |
| 📜 技能卡 | 10% | 每次五连最多 1 张，一次性使用 |
| 💰 金币 | ~25% | 100–1000 金币直接入账 |
| ⚔️ 将士 | ~27% | 100–1000 兵力补充到首城 |

## AI 机器人

等待阶段可添加 AI 机器人，共 4 种策略风格：

| 类型 | 风格 |
|------|------|
| 🔱 吕布型 | 好战嗜杀，优先进攻 |
| 📜 诸葛型 | 稳扎稳打，先发展后进攻 |
| 👑 曹操型 | 攻守兼备，伺机而动 |
| ⚔️ 张飞型 | 全力扩张，速战速决 |

## 存档系统

游戏中可随时存档，记录所有玩家状态、城池、武将和回合信息。加载存档后，所有玩家需用原名字重新加入房间。
