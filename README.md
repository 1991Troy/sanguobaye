# 🏯 三国霸业

多人联机回合制策略网页游戏，2-4人在线对战，争夺三国天下。

## 游戏入口

打开网页 → 输入名号 → 创建房间或加入房间 → 等人齐了点"开始游戏"。

把房间号（4位字母数字）发给朋友，他们输入房间号就能加入。手机电脑都能玩。

## 游戏规则

### 目标
占领全部城池，或者30分钟时间到后按得分结算（城池数占70%，兵力占30%）。

### 回合流程
每个回合你有60秒，可以做以下操作（不限次数）：

1. **五连抽卡** — 每回合免费抽一次，可能抽到武将、技能卡、金币或兵力
2. **出征** — 选择出发城池、目标城池、派遣武将和出兵数量，发起攻击
3. **全军出击** — 一键派出城池全部兵力（自动保留500兵守城）
4. **征兵** — 花金币在自己的城池招募士兵（1000兵 = 2000金）
5. **使用技能卡** — 使用抽到的技能卡获得战术优势
6. **结束回合** — 主动结束，轮到下一个玩家

### 地图
24座城池，基于三国真实地理分布。城池之间有固定路线，只能沿路线攻击（除非有特殊技能卡）。

每回合开始时：
- 每座城池产出1000金币
- 每座城池自动恢复500兵力（上限30000）

### 战斗系统
- 出征时城池至少保留500兵力
- 武将提供攻击/防御/谋略加成：攻击力影响进攻，防御力影响守城，谋略减少己方兵损
- 战斗结果有随机浮动（±20%）
- 攻城胜利后，攻方武将自动驻守目标城池
- 击败敌方城池可俘获驻守武将（未战死的武将归入攻方）
- 战败方武将有概率战死（金卡15%、紫卡25%、蓝卡40%）
- 武将名字会显示在城池地图上

### 抽卡系统

每个武将全局唯一，一旦被某位玩家抽到，其他玩家不会再抽到同名武将。

| 类型 | 概率 | 说明 |
|------|------|------|
| 🥇 金卡武将 | 5% | 吕布、关羽、马超、张飞、诸葛亮、孙权、曹操、刘备 |
| 🟣 紫卡武将 | 15% | 赵云、周瑜、典韦、司马懿、甘宁、太史慈 |
| 🔵 蓝卡武将 | 15% | 黄忠、孙策、张辽、徐晃、魏延、黄盖 |
| 📜 技能卡 | 10% | 每次五连最多1张，一次性使用 |
| 💰 金币 | 25% | 100-1000金币直接入账 |
| ⚔️ 将士 | 30% | 100-1000兵力补充到首城 |

## 本地运行

```bash
npm install
npm start
```

打开 `http://localhost:3000` 即可。

## 技术栈

Node.js + WebSocket + Canvas，无框架依赖。
